HTML5 e CSS

HTML5

O HTML5 é a mais recente versão da linguagem de marcação HTML e inclui muitos recursos novos e úteis. Abaixo está uma tabela de elementos semânticos HTML5 que foram apresentados em aula.

Elemento Usado? Descrição
Elemento em bloco ✔ Utilizei-o para criar uma divisão de conteúdo e aplicar CSS.
span ✔ Utilizei-o para alterar e personalizar texto.
article ❌ Ainda não utilizei este elemento.
section ✔ Utilizado para definir uma seção de conteúdo.
aside ✔ Utilizado para definir uma seção lateral, como uma barra lateral.
footer ✔ Utilizado para definir o rodapé de uma página.

CSS

O CSS é uma linguagem de estilo usada para descrever a apresentação de um documento HTML. Abaixo está uma tabela de tipos de seletores CSS.

Seletor Usado? Descrição
Seletor de elemento ✔ Seleciona todos os elementos com o nome especificado.
Seletor de classe ✔ Seleciona todos os elementos com a classe especificada.
Seletor de ID ✔ Seleciona um elemento com o ID especificado.
Seletor de atributo ❌ Ainda não utilizei esse seletor.
Seletor de descendente ✔ Seleciona todos os elementos que são descendentes diretos ou indiretos de um elemento especificado.
Seletor de filho ✔ Seleciona todos os elementos que são filhos diretos de um elemento especificado.

Abaixo está uma tabela de combinadores de seletores CSS.

Combinador Usado? Descrição
E F ✔ Seleciona todos os elementos F que são descendentes diretos ou indiretos de um elemento E especificado.
E > F ✔ Seleciona todos os elementos F que são filhos diretos de um elemento E especificado.
E + F ❌ Ainda não utilizei este combinador.
E ~ F ❌ Ainda não utilizei este combinador.
